あみたろの声素材工房

あみたろ様

はじめまして。Minecraft Paperプラグイン「MahjongPaper」の開発で、あみたろの声素材工房様の音声素材を使用させていただきましたので、ご報告いたします。

作品名: MahjongPaper
種別: Minecraft Paperプラグイン / CraftEngine向けリソースバンドル
公開日: [公開日]
公開URL: [正式な配布URLまたはリリースURL]
リポジトリURL: https://github.com/Arbousier1/MahjongEngine

使用用途:

Minecraft上で遊べる麻雀テーブルの演出音として使用しています。プレイヤーの「チー」「ポン」「カン」「リーチ」「ロン」操作時に、短い女声アクション音として再生されます。作品内容は全年齢向けです。

使用させていただいた音声素材:

- `chii_01.wav` -> `reaction_chi.ogg`
- `pon_01.wav` -> `reaction_pon.ogg`
- `kan_01.wav` -> `reaction_kan.ogg`
- `ri-chi_01.wav` -> `riichi.ogg`
- `ron_01.wav` -> `round_win.ogg`

配布形態:

プラグインに同梱されるCraftEngine向けリソースバンドル内に、Minecraft resource pack用の音声ファイルとして含めています。サーバー導入時にCraftEngineがリソースをまとめ、プレイヤーへ配信する形になります。

編集内容:

Minecraft resource packで使用できるよう、Ogg Vorbis形式への変換、音量調整、前後の無音カット、短いフェード処理を行いました。セリフ内容そのものの改変や、複数セリフの切り貼りによる内容変更は行っていません。

クレジット表記:

配布物内の `resourcepack/ATTRIBUTION.md` に、以下のクレジットと規約リンクを掲載しています。

```text

```
